# Talking Rabbitt 🐰

Talking Rabbitt is a conversational analytics prototype that allows users to upload business data and ask questions to generate insights and visualizations.

## Features
- Upload CSV sales dataset
- Ask natural language questions about the data
- Automatic insights and charts
- Demonstrates the "talking to data" concept

## Example Questions
- Which product has the highest revenue?
- Which product sold the most units?
- Show revenue by region
- Show revenue by category

## Tech Stack
- Python
- Streamlit
- Pandas

## How to Run Locally

Install dependencies

pip install -r requirements.txt

Run the app

python -m streamlit run app.py

## Dataset

Sample ecommerce sales dataset included in the repository.
